# PSVRCinemaUtil
Command line tool enables Cinema Mode for PSVR USB connected to macOS

Mouse Cursor is controlled by head movement.

Controls:

Volume +: Mouse Click

Volume -: Mouse Cursor Enabler while pushed

Mute: Re-center view and cursor
